Here is what happens inside the app:

- You can view this data clearly and easily.

This all happens within seconds, keeping you up to date.

## ‚öôÔ∏è Setup Instructions (For Advanced Users)

If you want to run the Weather-API app from source code, follow these steps. Otherwise, you can just download and install the ready app from the link above.

1. **Download the source code**

   You can clone the source repository or download the source files from GitHub.

2. **Install Node.js**

   Make sure you have Node.js installed on your computer. Visit [nodejs.org](https://nodejs.org/) to download and install it.

3. **Install dependencies**

   Open a command prompt or terminal in the source folder and run:

   ```
   npm install
   ```

4. **Get an API key**

   Sign up at [OpenWeatherMap](https://openweathermap.org/api) to get a free API key. This key allows the app to access weather data.

5. **Add your API key**

   Create a file named `.env` in the app root folder and add this line:

   ```
   VITE_WEATHER_API_KEY=your-api-key-here
   ```

   Replace `your-api-key-here` with the key you received.

6. **Run the app**

   In the terminal, run:

   ```
   npm run dev
   ```

   The app will open in your browser at `http://localhost:5173`.
